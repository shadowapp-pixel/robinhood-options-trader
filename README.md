# Robinhood Options Trader

A web-based application that provides live trade suggestions for Robinhood option contracts using statistical analysis, economic trends, and technical analysis.

## Features

- **Live Trade Suggestions**: Get real-time trade recommendations for any option contract
- **Statistical Analysis**: Advanced analysis based on market data and patterns
- **Economic Indicators**: Integration with national and local economic trends
- **Technical Analysis**: Chart patterns and technical indicators
- **Timeframe Breakdown**: Suggestions organized by contract length
- **Entry/Exit Strategies**: Specific price points and exit strategies

## Tech Stack

- **Frontend**: Next.js 15, React 19, TypeScript
- **Styling**: Tailwind CSS, shadcn/ui components
- **Backend**: Next.js API Routes
- **Database**: PostgreSQL with Prisma ORM
- **Charts**: Recharts for data visualization
- **UI Components**: Radix UI, Lucide React icons
